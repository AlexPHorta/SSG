# Issue Tracking

- Create the skeleton of the main app:
	- Run the app;
	- It will prompt for:
		- A folder name to place the website's files (config, paths, template etc);
		- The project's name;
		- The author's name;
		- The website's URL;
		- It will create the following directory structure:
			- project/
				- content/
				- template/
				- output/
				- settings.toml
				- paths.toml
